우분투 개요

Ubuntu는 오픈 소스 운영 체제(OS)이다.

우분투는 남아프리카 공화국의 줄루어로 “헌신, 다른 사람과의 관계"에 초점을 맞춘 철학적 의미를 가지고 있으며, 2004년에 오픈 소스 데비안 GNU/리눅스의 파생 배포판으로 출시되었다.

Ubuntu는 영국 Canonical사의 지원을 받으면서 커뮤니티 중심으로 개발이 진행되어 반년마다 최신 기술을 도입한 정기적인 버전 업 릴리스가 되고 있다.

우분투는 데스크톱과 넷북뿐만 아니라 서버에도 사용할 수 있다.1
그리고, 스마트폰용 Ubuntu를 탑재한 스마트폰도 판매되고 있다.

Ubuntu는 무료 보안 업데이트가 최소 9개월 동안 제공되는 일반 버전(데스크톱 및 서버)과 5년간 제공되는 장기 지원(LTS) 버전을 제공한다.
두 버전 모두 완전히 무료로 사용할 수 있으며 장기 지원(LTS) 버전으로의 업데이트도 무료이다.

Ubuntu에는 몇 가지 파생 버전이 있으며 공식적으로 데스크톱 환경과 목적에 따라 구성 패키지가 다른 여러 버전이 지원된다. 그 대표적인 버전은 다음과 같다.

  • Kubuntu : KDE를 표준 데스크탑 환경을 사용하는 버전
  • Lubuntu : LXDE를 표준 데스크탑 환경을 사용하는 버전
  • Xubuntu : Xfce를 표준 데스크탑 환경을 사용하는 버전
  • Ubuntu for Android : Android 단말에서의 동작을 목적으로 한 버전
  • Ubuntu Studio : 멀티미디어 편집에 특화된 버전

우분투는 간단한 화면, 완성도가 높고 세련된 사용자 인터페이스, 설치 및 업데이트가 쉽고, 완전히 무료로 이용할 수 있는, 무료 보안 업데이트가 붙는 등의 특징을 가지고, 전세계 2000만명 이상의 사용자가 많이 사용하고 있는 인기 1위인 Linux 배포판이다.
한국에서도 우분투의 주목도, 이용률이 높다.

  • 세계에서의 인기도의 동향(Ubuntu, Debian, Fedora, CentOS, openSUSE)
  • 한국에서의 인기도의 동향(Ubuntu, Debian, Fedora, CentOS, openSUSE)

우분투의 주요 특징

주요 특징은 다음과 같다.

특징 설명
직관적인 조작 세련되고 심플한 인터페이스이므로 초보자도 사용하기 쉽고 직관적인 조작이 가능
즉시 사용 가능 Live CD 환경에서 재부팅할 필요 없이 하드 디스크에 Ubuntu를 설치할 수 있으므로 기본 설정 작업을 완료하고 즉시 사용할 수 있다.
보안 대응 임베디드 방화벽, 안티바이러스, 피싱 대책이 제공되며, 자동 보안 업데이트가 제공되는 장기 지원(LTS)도 사용 가능
충실한 기능 기본적으로 브라우저(Mozilla Firefox), 메일 소프트(Mozilla Thunderbird), 오피스 스위트(LibreOffice), 음악, 비디오, 사진 편집, 캘린더 등의
애플리케이션이 설치되며 애드온에서 사용할 수 있는 무료 또는 유료 애플리케이션이 많다.
그리고, 패키지 수는 47,000 이상라고 한다. (https://packages.ubuntu.com/kinetic/allpackages)
설치 및 업데이트가 용이 데스크톱 버전은 그래픽 설치 프로그램이 있어 설치가 빠르고 간단하다.
보안 수정이나 버전 업그레이드 등의 업데이트는 업데이트 영역에 표시되는 것을 클릭하는 것만으로 업데이트할 수 있다.
호환성 MP3 플레이어, 카메라 및 프린터 연결을 위한 플러그인과 Microsoft Office 문서와 호환되는 오피스 제품군 제공
다국어 대응 40개 이상의 언어로 이용 가능하다.
반년마다 릴리즈 반년마다 정기적인 버전 업 릴리스가 제공되어 항상 최신 기술을 받아 볼 수 있다
비용 효과 우분투는 라이센스 비용이 들지 않기 때문에 상용 제품에 비해 대폭 도입 비용 절감을 할 수 있다.

우분투 라이센스

우분투 라이센스는 특정 라이센스 이름이 정의되지 않다. 이는 우분투가 수천 개의 프로그램으로 구성되어 있으며 각 프로그램이 별도의 라이센스에 따라 출시되기 때문이다.

우분투의 기반인 데비안 GNU/리눅스는 데비안 프리 소프트웨어 가이드라인(DFSG)에 따라 소프트웨어의 확장, 개조, 수정, 재배포를 자유롭게 인정한다.
그 파생 프로젝트인 Ubuntu도 Debian Free Software Guideline(DFSG)과 공통된 내용을 일부 포함하지만 Debian Free Software Guideline(DFSG)에 따르는 것은 명시되지 않다.

또한, Ubuntu의 라이센스는 GPL이라고 기재되어 있는 경우가 있지만, Ubuntu를 구성하는 Linux 커널등의 라이센스가 GPL인 것만으로, Ubuntu 프로젝트의 페이지 에서는 Ubuntu의 라이센스가 GPL이라고 하는 명시되어 있지 않다.

Ubuntu 프로젝트 페이지 에서는 Ubuntu 프로젝트가 기본적으로 UbuntuOS에 포함시킬 소프트웨어를 결정할 때 따르는 프로세스를 설명하는 라이센스 정책으로 다음과 같은 내용을 게재하고 있다.
기본적으로 설치되는 모든 응용 소프트웨어는 무료 소프트웨어이다.

  • 우분투는 다양한 라이선스 프로그램 및 문서의 모음으로, main, restricted, universe, multitiverse의 네 가지 주요 그룹 또는 구성 요소로 구성된다.
  • 우분투의 주요 컴포넌트에 포함된 모든 애플리케이션은 소스 코드를 포함해야 하며, 동일한 라이센스 하에서 변경된 사본의 변경 및 배포를 허용해야 한다.
  • main, restricted 컴포넌트에 포함된 모든 어플리케이션에 있어서, 재배포를 허가하고, 재배포나 변경에 수반하는 로열티 요구를 금지하며, Ubuntu에 고유한 라이센스하에서의 배포를 금지한다.

main, restricted, universe, multitiverse 각 그룹의 차이

리포지토리 유지보수 소스 코드 LTS에서 지원
main Canonical* 있다 있다
restricted Canonical* 없다 없다
유니버스 우분투 개발자 없다 없다
Mulitiverse 외부 개발자 없음 없음
  • Canonical: Ubuntu의 개발 지원, 자금 원조를 실시하고 있는 기업. 우분투의 창립자 마크 셔틀 워스에 의해 설립되었다.
  • Ubuntu는 Canonical Ltd.의 등록 상표이다.

그밖에 우분투 라이선스 정책에 대한 자세한 내용은 이 페이지를 참조하여라.

    • 무료 설치가 되지 않는다.
      • 메이커에 따라서는 Windows Office가 무료 설치가 되기 한다.

    • 그 외, 무료 오피스 스위트의 이용도 가능하다.
