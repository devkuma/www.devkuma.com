알고리즘

  • 자료구조에 쌓인 데이터를 활용해 어떤 문제를 해결하기 위한 여러 동작들의 모임이다.
  • 문제를 해결하는 절차를 말한다.
  • 입력, 출력, 명백성, 유효성, 효과성을 만족해야 한다.
  • 알고리즘은 논리이며 수학이고, 실질적인 개발에 적용되는 기초적인 아이디어이다.
